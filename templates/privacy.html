Политика конфиденциальности

1. Общие положения

1.1. Настоящая Политика конфиденциальности (далее — «Политика») регулирует порядок обработки и защиты информации, которую Пользователь передаёт при использовании сервиса Secure VPN (далее — «Сервис»).

1.2. Используя Сервис, Пользователь подтверждает своё согласие с условиями Политики. Если Пользователь не согласен с условиями — он обязан прекратить использование Сервиса.

2. Сбор информации

2.1. Сервис может собирать следующие типы данных:

  • идентификаторы аккаунта Telegram (ID пользователя, никнейм);
  • техническую информацию (IP-адрес, данные о браузере, устройстве для обеспечения работы);
  • данные, необходимые для обработки платежей (ID заказа, сумма, статус).

2.2. Сервис не требует и не хранит паспортные данные, документы, фотографии или другую личную информацию, кроме минимально необходимой для технической работы.

3. Использование информации

3.1. Сервис использует полученную информацию исключительно для:

  • обеспечения работы функционала (предоставление VPN-доступа);
  • связи с Пользователем (уведомления о статусе платежа, техподдержка);
  • анализа и улучшения работы Сервиса (обезличенная статистика).

4. Передача информации третьим лицам

4.1. Администрация не передаёт полученные данные третьим лицам, за исключением случаев:

  • если это требуется по закону;
  • если это необходимо для исполнения обязательств перед Пользователем (например, обработка платежа через платёжную систему Platega);
  • если Пользователь сам дал на это явное согласие.

7. Изменения в Политике

7.1. Администрация вправе изменять условия Политики. Актуальная версия всегда доступна на этой странице.

7.2. Продолжение использования Сервиса после внесения изменений означает согласие Пользователя с новой редакцией Политики.

Контакт для вопросов: @sequrevpn_help (Telegram)
